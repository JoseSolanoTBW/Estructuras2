Un B-tree es una estructura de datos que consta de nodos ordenados en un árbol balanceado. Cada nodo contiene keys (los números que se observan) e hijos (los nodos debajo del mismo).

Los nodos estan ordenados por el lado izquierdo, en el medio, o en la derecha dependiendo si sus claves(keys) son menores, intermedios o mayores que las claves de los padres.

La raíz es el nodo top, nodos internos están en el medio, y nodos hoja están al final. Usando esta terminología, árbol-b con un orden m válido sigue las siguiente reglas:

  • Cada nodo tiene como máximo n hijos.
  • Cada nodo interno tiene al menos ceil( m / 2 ) hijos.
  • El nodo raíz tiene al menos 2 hijos si no es una hoja.
  • Un nodo no hojan contiene hijos n-1 claves(keys).
  • Todas las hojas tiene un mismo nivel

Los B-trees permite realizar busquedas por valor y nos permite realizar de forma rapida las búsquedas de valores en bases de datos y file systems.
